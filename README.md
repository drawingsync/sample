# Historical Sample Drawings

Sample drawings from releases of AutoCAD in the early top mid 1980s

You may open these drawings with the file open dialog box.  This is unlike the releases in which the drawings were created which required you to exit the current drawing and load a special screen that forced you to type in the full path name of the drawing.

These are AutoCAD sample drawings created in the mid 1980s.  At that time these drawings were used to demonstrate that AutoCAD was as powerful as mainframe CAD systems.

Today, AutoCAD users regularly work with drawings 10,000 times larger then these drawings!

Several of these drawings were so old that the original files cannot be read by AutoCAD Release 14.  We opened these files in intermediate releases such as Release 2.52.  This release updated the file to a format that can be read by AutoCAD Release 14.

The lesson that can be learned from this experience is that, from time to time, very old computer data should be loaded and "freshened" up.
